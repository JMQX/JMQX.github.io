Axios(JS HTTP库/Ajax库)基本用法

2017-04-22

特征

vue更新到2.0之后，作者就不再对vue-resource更新了，而是推荐使用Axios。Axios的特征是：

  • 在浏览器中发送 XMLHttpRequests 请求
  • 比Jquery轻量，但处理请求不多时，可以使用
  • 基于Promise语法标准
  • 支持NodeJs
  • 拦截请求和响应
  • 自动转换JSON数据
  • 客户端支持保护安全免受 XSRF 攻击
